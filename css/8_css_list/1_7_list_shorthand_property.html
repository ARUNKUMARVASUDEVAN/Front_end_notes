List style - shorthand

Three values passed

  • The item marker is an image, position as outside. In absence of image, the marker will be a circle. Try not loading the image and see the type as circle.
  • It is used in cases where we need not follow a sequence.

Two values passed

  • The item marker is square, with style position as inside and no image.
  • It is used in cases where we need not follow a sequence.

One value passed

  • The item marker is disc, with style position as outside (default) and no image (default - none).
  • It is used in cases where we need not follow a sequence.
